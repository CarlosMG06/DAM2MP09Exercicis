Xifrat RSA per arxius

Eina senzilla per xifrar/desxifrar arxius petits amb claus RSA PEM. Ús educatiu.

Descargar .deb

Advertència

RSA directe és un algoritme bàsic d'encriptació i no és segur per a dades reals, públiques i sensibles. En producció, fes servir xifratge adequat com el que proporciona l'eina gpg.



Ús rápid

  • Instal·la el paquet i executa'l.
  • Usa claus públiques per xifrar i claus privades per desxifrar.


Generar claus

El programa només es compatible amb claus de format PEM PKCS#1 o PKCS#8.
Per generar claus PKCS#1, utilitza aquestes dues comandes: 


    openssl genrsa -out nom_clau_privada.pem 2048
    openssl rsa -pubout -in nom_clau_privada.pem -out nom_clau_publica.pem
